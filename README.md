# automatic_file_organizer

it is a python projec that automatically organises file into folder based on thier extensision 

#Features
automactillay creates a floder a and relavant files to it
 #how will it works
 --- you will upload a zip file(single directory) thta has multiple file without anything
 ---our gradio app will place each file into its right extensison 
 ---and return you a zip(organsized)
# requirements
     python (>=3.9)
     pip install gradio
     pip install gradio_client
